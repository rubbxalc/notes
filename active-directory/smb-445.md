# SMB - 445

## Uso de crackmapexec

### Escaneo básico
```null
crackmapexec smb 10.10.10.192
```

### Validación de credenciales
```null
crackmapexec smb 10.10.10.192 -u support -p '#00^BlackKnight'
```
